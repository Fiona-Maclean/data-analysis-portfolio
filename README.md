# Data Analysis Portfolio – Fiona Maclean

This repository contains selected data analysis projects demonstrating skills in Excel and Power BI.

## Projects

### 1. Hospital Data Analysis (Excel)
- Data structuring, formulas, and pivot tables
- Synthetic and anonymised dataset
- File: `hospital-excel/Fiona_Maclean_Hospital_Data_Excel.xlsx`

### 2. Hospital Referrals Dashboard (Power BI)
- Interactive dashboard showing referral trends and patient engagement
- Includes slicers for hospital and month
- Files: `hospital-powerbi/Fiona_Maclean_Hospital_Data_PowerBI.pbix` 

### 3. Postgraduate Computing Courses in Scotland (Power BI)
- Analysis of postgraduate computing courses (2014–2024)
- Stacked bar chart and trend lines for courses and student enrolments
- Slicers for university and course
- Source: HESA Table 49 (computer games courses excluded)
- Files: `hesa-powerbi/Fiona_Maclean_HESA_Postgrad_Computing.pbix`
